# homebrew-carbon-aware-tools

[Homebrew Tap](https://docs.brew.sh/Taps) for publishing formulae for carbon aware CLI tools.

e.g. grid-intensity

```
$ brew install thegreenwebfoundation/carbon-aware-tools/grid-intensity
```
